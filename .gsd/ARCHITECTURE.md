## Overview
MarketMind AI is a comprehensive stock market prediction platform leveraging machine learning and real-time data analysis. It mimics a microservices architecture with distinct services for data ingestion, machine learning, backend API, and frontend UI.

## Components

### Frontend
- **Purpose**: User interface for dashboard, portfolio management, and visualizations.
- **Location**: `marketmind-ai/frontend`
- **Tech**: React, Vite, TypeScript, TailwindCSS (planned).

### Backend
- **Purpose**: API gateway, user authentication, and business logic.
- **Location**: `marketmind-ai/backend`
- **Tech**: Node.js, Express, TypeScript.

### ML Services
- **Purpose**: Data ingestion, model training, and inference.
- **Location**: `marketmind-ai/ml-services`
- **Components**:
    - `data_pipeline/stock_fetcher.py`: Historical data ingestion.
    - `data_pipeline/real_time_fetcher.py`: WebSocket client.
    - `news_pipeline/scraper.py`: News aggregation.
    - `data_pipeline/data_quality.py`: Data integrity checks.

### Infrastructure
- **Location**: `marketmind-ai/infrastructure`
- **Services**: TimescaleDB, MongoDB, Redis, Kafka, Zookeeper, PostgreSQL.

## Data Flow
1.  **Stock Data**: `StockFetcher` -> `yfinance` -> `TimescaleDB (hypertable)`.
2.  **News Data**: `NewsScraper` -> `RSS/API` -> `MongoDB`.
3.  **Real-time**: `RealTimeFetcher` -> `WebSocket` -> `TimescaleDB`.
